===================================
COLLISION MECHANICS
===================================

COLLISION RADIUS: 0.5 units
- Objects within 0.5 units = COLLISION
- Distance > 0.5 units = NO COLLISION

COLLISION DETECTION:
1. Player tries to move to (newX, newY)
2. Check distance from (newX, newY) to ALL objects
3. If any distance < 0.5 â†’ BLOCKED
4. Otherwise â†’ MOVE ALLOWED

TECHNICAL DETAILS
===================================

Algorithm: Distance-based Collision Detection
- O(n) per movement (check all objects)
- For 10 objects: ~10 distance calculations
- Fast enough for turn-based games

Formula: sqrt((x2-x1)Â² + (y2-y1)Â²) < 0.5

Implementation:
- GameManager::canMoveTo() - preventive checking
