===================================
HOW PROJECTILES WORK
===================================

INSTANT HIT MODEL:
1. Player fires Bow
2. Arrow projectile created at player position
3. Arrow "flies" toward target (visual)
4. Arrow checks collision immediately
5. Arrow deals damage if hit
6. Arrow marked as expired
7. Arrow removed from game
All in ONE turn!

PROJECTILE LIFECYCLE:
Create ‚Üí Process ‚Üí Damage ‚Üí Cleanup

===================================
TECHNICAL DETAILS
===================================

PROJECTILE PROPERTIES:
- Damage: Same as weapon power (Bow = 1)
- Owner: Player who fired it
- Target: Character being attacked
- HasHit: Tracks if already dealt damage

COLLISION DETECTION:
- Projectile at player position
- Target at enemy position
- Distance check (collision detection)
- If hit: apply damage

CLEANUP:
- After projectile.update(), hasHit = true
- Cleanup phase calls removeExpiredProjectiles()
- unique_ptr automatically deletes projectile
- No memory leaks!
